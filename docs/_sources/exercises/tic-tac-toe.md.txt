# Tic-Tac-Toe

Code an interactive, two-player game of Tic-Tac-Toe. You'll use a two-dimensional list of `str`s.

## Starter Code
* [tic_tac_toe.py](../_static/examples/tic_tac_toe.py)

```
(...a game already in progress)

	O X X
	  X O

	X O O

	  X O

	X O O

	X X O

```
