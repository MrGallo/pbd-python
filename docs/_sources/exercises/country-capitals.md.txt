# Country Capitals

Download both files into the same folder.

```eval_rst
* :download:`country-by-capital-city.json <examples/country-by-capital-city.json>` - `source <https://raw.githubusercontent.com/samayo/country-json/master/src/country-by-capital-city.json>`_
* :download:`country_capitals.py <examples/country_capitals.py>`
```

## What you should do
1. Modify the `country_capitals.py` code to display the name and capital city for every country as shown below.
    ```
    ...
    ...
    Country: Western Sahara
    Capital: El-Aai

    Country: Yemen
    Capital: Sanaa

    Country: Zambia
    Capital: Lusaka

    Country: Zimbabwe
    Capital: Harare
    ```
2. Convert the loading of the JSON file into a function called `load_countries`. It will have one parameter, the name of the file to load. It will return a list of country dictionaries.
