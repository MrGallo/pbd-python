19.1. Basic Lists 0¶

Create an empty list. Append to the array the number -113 ten times using the list.append() method. Then display the contents of the list on the screen. One element at a time.

Do not use a loop. Also, do not use any variable for the index; you must use literal numbers to refer to each slot.
