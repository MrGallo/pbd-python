Tic-Tac-Toe¶

Code an interactive, two-player game of Tic-Tac-Toe. You’ll use a two-dimensional list of strs.

Starter Code¶

  • tic_tac_toe.py

(...a game already in progress)

	X   O
	O X X
	  X O
 
'O', choose your location (row, column): 0 1

	X O O
	O X X
	  X O
 
'X', choose your location (row, column): 2 0

	X O O
	O X X
	X X O

The game is a tie.
