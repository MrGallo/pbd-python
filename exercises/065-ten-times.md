# Ten Times

Write a program that prints the important phrase "Hello, World!"
on the screen ten times. Use a `for` loop to do it.

Name the file: `065_ten_times.py`

```
Hello, World!
Hello, World!
Hello, World!
Hello, World!
Hello, World!
Hello, World!
Hello, World!
Hello, World!
Hello, World!
Hello, World!
```

If you want, you can number the lines of output like so:

```
1. Hello, World!
2. Hello, World!
3. Hello, World!
4. Hello, World!
5. Hello, World!
6. Hello, World!
7. Hello, World!
8. Hello, World!
9. Hello, World!
10. Hello, World!
```
